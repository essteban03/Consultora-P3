**Institución:** Universidad de las Fuerzas Armadas ESPE

## Descripción

Landing page profesional para NexoDigital Consulting, una empresa ficticia de consultoría en transformación digital enfocada en pequeñas empresas. La página comunica los servicios de digitalización de procesos, automatización y soluciones empresariales, e incluye un formulario visual de contacto comercial.

## Tecnologías utilizadas

- HTML5 semántico
- CSS3 (archivos externos, enfoque mobile first)
- Bootstrap 5.3.3
- Bootstrap Icons 1.11.3
- Google Fonts (Poppins, Inter)

## Instrucciones para abrir la página

1. Clonar o descargar el repositorio.
2. Abrir el archivo `index.html` directamente en el navegador, o servirlo con la extensión "Live Server" en el editor de código.
3. No requiere instalación de dependencias adicionales (Bootstrap y los íconos se cargan por CDN).
